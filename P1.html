This insight is drawn from a professional survey on some Data related job roles in a region.
I cleaned and Visualized the data set using Power Bi. The data sets included Data job roles,
Average salary by Job title, favorite programming language, Difficulties to break into data Job roles or get started.

From the report 630 persons participated in the survey of an average age 29.87 years.
Data Scientist earn the Average highest salary and Database Developers/Students earn the least.

Python had the highest record for the most favourite programming language with 420 participant out of 630.
Data analyst leads the table amongst other data Job roles.
